# Document Ingestion Plan

## Current indexed support themes
- login and account access
- billing and subscription
- user invitations and permissions
- notifications and email delivery
- performance and slow loading

## Next ingestion goals
We want to make the knowledge base more realistic by adding:
- longer support articles
- FAQ-style documents
- release-note style documents
- help-center style documents
- overlapping topic documents

## First ingestion batch
### Batch 1: Help-center style docs
- account security and password reset
- billing FAQ
- invitation troubleshooting FAQ
- notification troubleshooting FAQ
- performance troubleshooting FAQ

### Batch 2: Release-note style docs
- recent login changes
- billing portal improvements
- invitation flow update
- notification delivery fix
- dashboard loading improvements

### Batch 3: Mixed support docs
- short FAQ pages
- troubleshooting guides
- policy/limitations pages

## Rules for ingestion
- keep topics realistic
- allow some overlap between documents
- do not try to keep wording too perfect
- test retrieval after each batch
- rerun eval after indexing each batch
